# 更新日志

## 2026.07.22

- 修复电量与温控同时触发后提前恢复充电、继而再次停充的问题；现在会记录停充原因，并等待所有条件恢复
- 统一换算不同设备上摄氏度、十分之一度、百分之一度和千分之一度的电池温度，避免误触发温控
- 安装前增加音量键确认，按音量下或 20 秒未确认会安全取消
- 更新同 ID 模块时可选择安全迁移旧配置或使用新版默认配置；迁移时仅保留合法字段和数值
